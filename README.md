Crypto Portfolio Dashboard
A full-stack cryptocurrency tracking and portfolio management application built with modern web technologies and real-time financial data integration.
 Live Demo:
https://crypto-app-32c98.web.app⁠
 Tech Stack
Frontend: React (Vite), TailwindCSS
Authentication: Firebase Authentication (Google OAuth)
Database: Firestore (NoSQL)
Charts & Data: CoinGecko API (Live Market Data)
Hosting: Firebase Hosting
Development Environment: Firebase Emulator Suite
 Features
 Secure Google Sign-In   Authentication
 Live Cryptocurrency Prices
 Candlestick Charts
 Volume Charts
 Multi-Asset Comparison Charts
 User-Specific Portfolio Storage
 Production Deployment
 Architecture Overview
User authentication handled via Firebase Auth.
Each authenticated user has a unique Firestore document (UID-based).
Portfolio data is stored securely per user.
Live market data fetched from CoinGecko API.
Charts dynamically update based on selected assets.
